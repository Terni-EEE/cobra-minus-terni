## About Cobra

Cobra is a Python IDE made by terni. that aims to not only fit advanced and intermediate Python 
programmers' needs, but also give beginners a special feature that converts text-based code to visual representations.

Currently, it is in Beta, so expect bugs and please notify ternitriad@gmail.com
